# Drew Bot

Drew is an all-in-one Discord bot focused on **security, automoderation, and visual content**. It is designed to provide **advanced raid protection**, efficient moderation tools, and entertainment features, especially aimed at **profiles and image-based content**.

---

## Main Features

### Security & Automoderation



### Content & Entertainment

* Image autoposting:

  * Avatars
  * Banners
  * Gangster style
  * Kawaii style

### Premium System

* **15-day free trial**
* **Freemium** business model

## Project Architecture

* **Language:** Python
* **Database:** MongoDB
* **Cache:** Redis
* **API:** TypeScript

## Installation

```bash
# Clone the repository
git clone https://github.com/neevets/drew-bot.git
cd drew-bot
```

```bash
# Install dependencies
pip install -r requirements.txt
```

Configure environment variables:

```env
DISCORD_TOKEN=""
DISCORD_PREFIX=""
BOT_OWNERS=
SENTRY_DSN=""
POSTGRES_URL=""
REDIS_URL=""
REDIS_TOKEN=""
BETTERSTACK_HEARTBEAT=""
```

---

## Usage

1. Invite the bot to your server
2. Grant the required permissions
3. Use the web panel or commands to configure security and autoposting

Example:

```text
/drew setup
```
